 Faça um algoritmo que solicite ao usuário 2 números, e escreva para o usuário uma resposta
informando qual dentre os dois números digitados é o maior. Faça o algoritmo nas suas 3
formas.

   1- Descrição narrativa

   1- Solicitar ao usuário que Informe um numero aleatorio
   2- Solicitar ao usuário outro numero aleatorio
   3- Caso o primeiro numero seja maior que o segundo, informar ao usuario: O primeiro numero digitado é maior, 
      Caso o segundo numero seja maior que o primeiro, informar ao usuario: O segundo numero digitado é maior, 
      Caso contrario informe: Os numeros são iguais

    2- Pseudocódigo

    var num1
    var num2

    readln('Informe um numero aleatorio', num1);
    readln('Informe um outro numero aleatorio', num2);

    if(num1 > num2)then
        writeln('O primeiro numero é o maior')
    elseif(num2 > num1)
        writeln('O segundo numero é maior')
    else 
        writeln('Os numeros são iguais')
